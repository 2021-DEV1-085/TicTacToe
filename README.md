# About : 
Kata / TicTacToe using Java 

# Technologies : 
Java 11  
Spring Boot 2.5.4  
Thymeleaf 2.5.4  

# Tools used :  
Intellij 2021 Community  

# To run this project : 
  1) maven clean install 
  2) Launch the Spring project in your IDE or with command line
  3) Visit http://localhost:8080/play
